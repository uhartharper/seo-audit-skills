# SEO Skills

Custom knowledge base for Claude Code SEO audits. Each skill is a Markdown file
loaded automatically when the relevant topic is invoked.

## Skills included

| Skill | Description |
|-------|-------------|
| `wordpress-divi` | WordPress + Divi: H1, CSS inline, JS defer, security, CWV |
| `wordpress-elementor` | WordPress + Elementor: LCP lazy load, WP Rocket conflicts, WooCommerce |
| `prestashop-seo` | PrestaShop: sitemap 404, CCC, URL structure, PHPSESSID, schema |
| `google-tag-manager` | GTM debugging: event not reaching GA4, Consent Mode v2, firing order |
| `ga4-analysis` | GA4: organic vs paid, attribution, DebugView, GSC integration |
| `se-ranking` | SE Ranking: rank tracking interpretation, Site Audit limitations |
| `screaming-frog` | Screaming Frog: crawl modes, JS rendering config, false positives |
| `semrush` | Semrush: Organic Research, Keyword Gap, data accuracy |

## Installation

Copy any skill folder to `~/.claude/skills/[skill-name]/SKILL.md`.
Claude Code loads skills automatically from that directory.

## Privacy

All knowledge is anonymized. No client names, domains, or personally
identifiable information. Compliant with GDPR / RGPD.

## Contributing

Enrich skills with real patterns as new issues are discovered.
Rule: knowledge enters anonymized — the pattern matters, not the source.
